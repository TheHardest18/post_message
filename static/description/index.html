Go to Sale →Reports → Reports Tss → Bonification Reports


How works?

The user enters the payroll period in MM / YYYY format for which they want to generate the report, once the period is set, they only have to click on the generate report button and a list will be displayed with all the payroll employees who have bonuses in Once the generate report is clicked, this period will be disabled and a new button "Download TXT" is enabled which allows downloading the report in txt format.
If you want to add an employee in that period already generated, you just have to click on edit and then add a line and fill in the form with the corresponding data.
Once the "Download TXT" button is clicked, the report changes to sent status and once there, no modification can be made.

The data types needed to generate this Tss Report are:

  • Type Document
  • Document number
  • Name
  • Surname
  • Second surname
  • Gender
  • Birthday
  • Bonification


Autodetermination v4.8

Go to Payroll →Reports → Reports Tss → Autodetermination Reports


How works?

The user enters the payroll period in MM / YYYY format for which they want to generate the report, Once the period has been established, you must select the type of file AM = monthly self-determination or AR = retroactive self-determination, once the corresponding fields are filled, you must click on generate report when you click and a list with all the payroll employees will be displayed When the report is generated, the generate report button will be deactivated and a new "Download TXT" button will appear. enabled that allows you to download the report in txt format.
If you want to add an employee in that period already generated, you just have to click on edit and then add a line and fill in the form with the corresponding data.
Once the "Download TXT" button is clicked, the report changes to sent status and once there, no modification can be made.

The data types needed to generate this Tss Report are:

  • Payroll key
  • Type Document
  • Document number
  • Name
  • Surname
  • Second surname
  • Gender
  • Birthday
  • Quotable salary
  • Voluntary contribution
  • ISR salary
  • Type of income
  • Others remunerations
  • RNC/ID. Withholding agent
  • Remuneration Other agents
  • Balance in favor of the period
  • Easter regalia(Balance 13)
  • Notice, severance pay, per diem and compensationfor work accidents
  • Alimony withholding
  • INFOTEP salary
